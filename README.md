## 📖 What is Bionic Reading?
Bionic Reading is a **psychological reading technique** invented by Swiss typographic designer **Renato Casutt**. It subtly **bolds the first few letters of each word**, creating artificial fixation points that guide the eye, improve focus, and boost comprehension.

## ✨ Features of the Oxide Userscript
- **Apple‑style Dark Mode** panel with glass‑morphism, smooth micro‑animations, and premium UI.
- **Dynamic configuration** – adjust bold strength, weight, dimming, and skip‑code‑blocks.
- **Cross‑site support** – works on any `http/https` page and on local `file://` documents.
- **Keyboard shortcut** – `Alt+Shift+B` toggles the effect instantly.
- **Sync settings** across browsers via Greasemonkey/Tampermonkey storage.

## 🚀 Installation
1. Install a userscript manager such as **Tampermonkey** or **Violentmonkey**.
2. Open the script URL below and click **Install**:
   - <https://github.com/Oxide5401/Bionic-Reader-Oxide/raw/main/Bionic%20Reading%20by%20Oxide%201.3.0.user.js>
3. The script runs on page load. Use the floating **FAB** in the bottom‑right to open the settings panel.

## 🛠️ Customisation
The settings panel (FAB) lets you fine‑tune:
- **Bold strength** – percentage of the word to bold (25‑65 %).
- **Bold weight** – 500 (Medium) to 900 (Black).
- **Dim un‑bolded text** – optional opacity dimming.
- **Skip code blocks** – avoid breaking code formatting.
- **Site‑specific enable/disable** – toggle per‑domain.

All changes are saved instantly via `GM_setValue` and are synced across tabs.
